# 🏠 Aura Home

## Aura Home Automation using Arduino IoT Cloud

Aura Home is an IoT-based smart home automation project that allows users to control home appliances remotely through the Arduino IoT Cloud. The system uses a Wi-Fi-enabled microcontroller and a relay module to switch appliances ON and OFF in real time.

## Features

* Remote appliance control
* Arduino IoT Cloud integration
* Control up to 4 appliances
* Real-time ON/OFF switching
* Wi-Fi connectivity

## Hardware

* Arduino IoT compatible board (ESP8266)
* 4-Channel Relay Module
* Wi-Fi Network
* Power Supply

## Software

* Arduino IDE
* Arduino IoT Cloud
* ArduinoIoTCloud Library
* Arduino_ConnectionHandler Library

## Working

1. Connect the board to Wi-Fi.
2. Sync with Arduino IoT Cloud.
3. Control appliances using the cloud dashboard.
4. Relays switch the connected devices ON or OFF.

## Applications

* Smart Home Automation
* Home Appliance Control
* IoT Learning Projects
